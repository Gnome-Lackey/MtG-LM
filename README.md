# MtGLM

## File Structure

```
] public - This is where all non-referenced static content is stored.
] src - This is where all our JavaScript is stored.
    ] assets
    ] components
        ] Common
        ] Form
        ] Hooks
        ] Navigation
        ] Views
    ] constants
        ] models
    ] models
    ] redux
        ] actions
        ] creators
        ] models
        ] reducers
    ] services
        ] models
    ] setup
    ] styles
    ] utils
] webpack
```

## Local development

Development build:

```bash
yarn start
```

Run tests:

```bash
yarn test
```

Build production bundle:

```bash
yarn build
```
